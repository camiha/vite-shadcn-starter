# vite-shadcn-starter

simple setup for Vite + shadcn/ui (w/TypeScript, Biome).  
In addition to the tutorial setup, we have set up a typography component and a color theme switch.

## setup
- `pnpm install`

## commands
- `pnpm dev`
- `pnpm build`
- `pnpm lint`
- `pnpm preview`
